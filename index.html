Boxmeup

Web and mobile application to help keep track of stuff

View the Project on GitHub boxmeup/Boxmeup

  • Download ZIP File
  • Download TAR Ball
  • Fork On GitHub

Benefits

  • Create containers and give them meaningful names to find them later through a list.
  • Search all of your containers for a specific item to find where it is physically located.
  • Add locations for containers to know their position in the world.
  • Print and scan QR codes with labels to place on your containers. Scanning the QR code with a smart phone will pull up the manifest of the contents of the container.
  • Android application allows scanning QR codes and product barcodes for simplistic item adding. (iOS app in the works)
  • Public API to have external application access and modify containers and items.
